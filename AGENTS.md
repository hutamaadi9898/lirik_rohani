Roles that help ship this app (human can play any/all):

- Product/IA: clarifies search UX, SEO requirements, admin permissions, and launches.
- Frontend: Astro + React islands, shadcn/ui styling, Framer Motion cues, accessibility/perf polish.
- Backend/Infra: Cloudflare Workers adapter, D1 schema/migrations (Drizzle), KV caching/audit log, FTS tuning.
- Data/Lyrics Editor: curates & validates songs, ensures clean formatting, approves releases.
- QA/Perf: tests search relevance/latency, link/SEO checks, cross-device smoke tests.

Coordination rule: after finishing any task, immediately update `plan.md` checkboxes and add a one-line entry in `progress.md` with date + what changed.
